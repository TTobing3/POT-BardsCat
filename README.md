### 주사위 게임 규칙

**운명의 눈** 주사위 게임 규칙 정리

[1] 패의 구분 (두 개의 주사위를 굴려 판별)

쌍둥이 : 두 주사위의 눈이 같음<br>
형제 : 두 주사위 눈이 1 차이<br>
친구 : 두 주사위 눈이 2 이상 차이<br>
원수 : 두 주사위가 (1,6) 또는 (6,1)<br>

[2] 패의 강함 순위 : 

쌍둥이 > 형제 > 친구 > 원수 > 쌍둥이 (순환 구조)

[3] 같은 계급 내 승패 판정

-주사위 숫자의 합이 높은 쪽이 승리

-합이 같다면 재경기
